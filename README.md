# Face Detection — минимальный проект

Простейший детектор лиц на OpenCV (Haar Cascade). Ловит лицо с веб-камеры,
из видеофайла или с одной картинки и рисует рамки вокруг найденных лиц.

Каскад Хаара выбран специально: он встроен в сам пакет `opencv-python`
(ничего дополнительно скачивать не нужно) и работает быстро даже на слабом
железе вроде Raspberry Pi — правда, ценой не самой высокой точности по
сравнению с DNN/MediaPipe-моделями. Для учебного проекта — то что нужно,
дальше легко заменить каскад на что-то более точное.

## Установка

```bash
python3 -m venv venv
source venv/bin/activate      # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

## Запуск

Веб-камера (по умолчанию камера с индексом 0):

```bash
python3 detect.py
```

Другая камера или видеофайл:

```bash
python3 detect.py --source 1
python3 detect.py --source path/to/video.mp4
```

Одна картинка:

```bash
python3 detect.py --source path/to/photo.jpg --image
```

Выход — клавиша `q` в окне с видео.

## Заметки по Raspberry Pi

- **USB-веб-камера** — работает без изменений, `cv2.VideoCapture(0)` увидит
  её как обычно.
- **Официальный Camera Module (CSI-шлейф)** — через `cv2.VideoCapture` она
  не всегда видна напрямую. Два рабочих варианта:
  1. Поставить `libcamera` + включить legacy-совместимость, чтобы Camera
     Module отображался как `/dev/video0`.
  2. Использовать библиотеку `picamera2`, захватывать кадры ей, а в
     `detect_faces()` передавать уже готовый numpy-массив кадра — сам код
     детекции трогать не придётся.
- **Производительность**: на Pi 3/4 в разрешении 640x480 Haar Cascade
  обычно даёт приемлемый FPS. Если тормозит — уменьшай кадр перед
  детекцией (`cv2.resize`) или увеличивай `minNeighbors`/`scaleFactor`.
- Ставь `opencv-python-headless` вместо `opencv-python`, если Pi без
  графического окружения и `cv2.imshow` не нужен (тогда рамки можно,
  например, только сохранять в файл или стримить по сети).

## Куда развивать дальше

- Заменить Haar Cascade на DNN-детектор (`cv2.dnn` + res10 SSD) — точнее,
  но чуть тяжелее.
- Добавить распознавание (не просто детекцию) через `face_recognition`
  или `dlib`.
- Прикрутить сохранение кадров с найденными лицами в файлы/базу.
